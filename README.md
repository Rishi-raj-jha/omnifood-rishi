# 🍽️ Omnifood

A modern, fully responsive landing page for **Omnifood**, an AI-powered healthy meal subscription service.

## 🚀 Live Demo

https://omnifood-rishi.netlify.app

## 📸 Preview

![Omnifood Preview](img/hero.png)



## ✨ Features

- Responsive design for all devices
- Sticky navigation
- Mobile navigation menu
- Smooth scrolling
- Hero section
- Meals section
- Testimonials
- Pricing plans
- Contact/Signup form
- Netlify Forms integration
- Custom Thank You page
- SEO optimized
- Lighthouse optimized
- Cross-browser compatible

The website is fully responsive and works on:

- Desktop
- Laptop
- Tablet

## 📬 Contact Form

The signup form is powered by **Netlify Forms**.

Features:
- Form validation
- Stores submissions in Netlify Dashboard
- Redirects users to a custom Thank You page after submission
